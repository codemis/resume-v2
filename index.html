About Me

Passion is the thing that will help you create the highest expression of your talent.

Larry Smith (Professor of Economics, University of Waterloo)

Passion is the reason I became a programmer. When I graduated college, I wanted to solve a real world problem that required programming skills. I did not have those skills, so using the internet I started to learn to program. After a year, I launched my first PHP based website to address the problem. As a result, I fell in love with programming and decided to change my career path. It was one of the best decision that I have made.

Many years later and I still love being a programmer! There is just something about taking an idea, drafting it out on paper, and then building it into existence.

I currently find myself working as the Project Director and Lead Developer for a ministry called Missional Digerati. I started the organization in 2013 in order to support Christian non-profits by developing mobile apps and websites that help them fulfill their unique mission. It has been an amazing opportunity that has given me a place to grow as a developer and a project lead.

Unreached of the Day Website

The Unreached of the Day website provides individuals an opportunity to pray for groups of people that have limited access to the Bible. The website was designed for people accessing the website in a low bandwidth internet situation. Using heavy caching, code splitting, three shaking and code compression, we deliver a fast user experience. The website is built using Webpack to deliver the fast response. It also provides daily reminders through browser based push notifications.
